Stage-opdracht: Molen

Deze opdracht test je css skills.

Het is de bedoeling dat je het beeld aan de rechterkant namaakt in de container links, met behulp van alleen css. Het enige bestand dat mag worden aangepast is molen-opdracht.css. Je mag geen elementen toevoegen in de html, met css mag je dus alleen de elementen gebruiken die al aanwezig zijn in de opdracht container. De animatie moet ook worden overgenomen. Het gebruik van plaatjes is niet toegestaan.

Tip voor de vorm van de basis van de molen: gebruik css borders!
